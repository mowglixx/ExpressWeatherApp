# ExpressWeatherApp

> Weather web app using express and handlebars

---

## Start

```sh
git clone https://github.com/mowglixx/ExpressWeatherApp.git
cd ExpressWeatherApp/
```

In `ExpressWeatherApp/` run

```sh
npm install
npm start
```
